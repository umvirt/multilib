10.1. Multilib Introduction

Now you will modify the new installation to add multilib support. This will not appear in a released LFS book, but was added here to show an example of how to do it.

The package selections listed here are the packages needed from LFS to continue on to building the necessary 32-bit xorg packages to run TeamViewer (the only 32-bit program I really need), but should also cover those needed (only of LFS) for Wine.

Create a temporary working directory to build a multilib enabled binutils and static gcc: 

mkdir /opt/temptools
export PATH_HOLD="${PATH}"
export PATH="/opt/temptools/bin:${PATH}"
