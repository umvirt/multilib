10.13. Acl-2.2.53 32-bit Library

The Acl package contains utilities to administer Access Control Lists, which are used to define more fine-grained discretionary access rights for files and directories. Here we install only the 32-bit version of the library provided by the package.

Approximate build time: less than 0.1 SBU
Required disk space: 4.9 MB

10.13.1. Installation of Acl

Prepare Acl for compilation: 

CC="gcc -m32" ./configure                \
    --prefix=/usr --libdir=/usr/lib32    \
    --disable-static --libexecdir=/usr/lib32

Compile the package: 

make

Install the package: 

make DESTDIR=$PWD/DESTDIR install
cp -Rv DESTDIR/usr/lib32/* /usr/lib32

Details on this package are located in Section 6.25.2, “Contents of Acl.”
