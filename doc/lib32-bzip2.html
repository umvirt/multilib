10.10. Bzip2-1.0.6 32-bit Library

The Bzip2 package contains programs for compressing and decompressing files. Compressing text files with bzip2 yields a much better compression percentage than with the traditional gzip. Here we install only the 32-bit version of library provided by the package.

Approximate build time: less than 0.1 SBU
Required disk space: 2.3 MB

10.10.1. Installation of Bzip2

The following command ensures installation of symbolic links are relative: 

sed -i 's@\(ln -s -f \)$(PREFIX)/bin/@\1@' Makefile

Prepare Bzip2 for compilation: 

make CC="gcc -m32" BIGFILES="" -f Makefile-libbz2_so
make clean

Compile the package: 

make CC="gcc -m32" \
     BIGFILES=""

Install the package: 

make PREFIX=$PWD/DESTDIR/usr install
cp -Rv DESTDIR/usr/lib/* /usr/lib32
cp -av libbz2.so* /usr/lib32

Details on this package are located in Section 6.21.2, “Contents of Bzip2.”
