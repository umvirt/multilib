10.9. File-5.34 32-bit Library

The File package contains a utility for determining the type of a given file or files. Here we install only the 32-bit version of libmagic provided the package.

Approximate build time: 0.1 SBU
Required disk space: 16 MB

10.9.1. Installation of File

Prepare File for compilation: 

CC="gcc -m32" ./configure \
    --prefix=/usr --libdir=/usr/lib32

Compile the package: 

make

Install the package: 

make DESTDIR=$PWD/DESTDIR install
cp -Rv DESTDIR/usr/lib32/* /usr/lib32

Details on this package are located in Section 6.12.2, “Contents of File.”
