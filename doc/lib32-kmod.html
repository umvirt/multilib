10.16. Kmod-25 32-bit Library

The Kmod package contains libraries and utilities for loading kernel modules. Here we install only the 32-bit version of the library provided by the package.

Approximate build time: 0.1 SBU
Required disk space: 12 MB

10.16.1. Installation of Kmod

Prepare Kmod for compilation: 

CC="gcc -m32" ./configure \
    --prefix=/usr         \
    --bindir=/bin         \
    --libdir=/usr/lib32   \
    --sysconfdir=/etc     \
    --with-xz             \
    --with-zlib           \
    --with-rootlibdir=/usr/lib32

Compile the package: 

make

Install the package: 

make DESTDIR=$PWD/DESTDIR install
cp -Rv DESTDIR/usr/lib32/* /usr/lib32

Details on this package are located in Section 6.46.2, “Contents of Kmod.”
