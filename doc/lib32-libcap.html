10.14. Libcap-2.25 32-bit Library

The Libcap package implements the user-space interfaces to the POSIX 1003.1e capabilities available in Linux kernels. These capabilities are a partitioning of the all powerful root privilege into a set of distinct privileges. Here we install only the 32-bit version of the library provided by the package.

Approximate build time: less than 0.1 SBU
Required disk space: 1.3 MB

10.14.1. Installation of Libcap

Prepare Libcap for compilation: 

sed -i '/install.*STALIBNAME/d' libcap/Makefile

Compile the package: 

make CC="gcc -m32"

Install the package: 

make lib=lib32 prefix=$PWD/DESTDIR/usr -C libcap install
cp -Rv DESTDIR/usr/lib32/* /usr/lib32

Details on this package are located in Section 6.26.2, “Contents of Libcap.”
