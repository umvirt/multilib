10.15. Xz-5.2.4 32-bit Library

The Xz package contains programs for compressing and decompressing files. It provides capabilities for the lzma and the newer xz compression formats. Compressing text files with xz yields a better compression percentage than with the traditional gzip or bzip2 commands. Here we install only the 32-bit version of the library provided by the package.

Approximate build time: 0.2 SBU
Required disk space: 15 MB

10.15.1. Installation of Xz

Prepare Xz for compilation: 

CC="gcc -m32" ./configure \
    --prefix=/usr \
    --disable-static \
    --libdir=/usr/lib32

Compile the package: 

make

Install the package: 

make DESTDIR=$PWD/DESTDIR install
cp -Rv DESTDIR/usr/lib32/* /usr/lib32

Details on this package are located in Section 6.45.2, “Contents of Xz.”
