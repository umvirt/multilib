10.8. Zlib-1.2.11 32-bit Library

The Zlib package contains compression and decompression routines used by some programs. Here we install only the 32-bit version of the library.

Approximate build time: less than 0.1 SBU
Required disk space: 4.5 MB

10.8.1. Installation of Zlib

Prepare Zlib for compilation: 

CC="gcc -m32" ./configure \
    --prefix=/usr --libdir=/usr/lib32

Compile the package: 

make

Install the package: 

make DESTDIR=$PWD/DESTDIR install
cp -Rv DESTDIR/usr/lib32/* /usr/lib32

Details on this package are located in Section 6.11.2, “Contents of Zlib.”
